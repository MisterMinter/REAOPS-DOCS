# Telegram Bot Guide

The Telegram bot is the primary way to interact with REAOPS.

## Getting Started
1. Find the bot on Telegram
2. Send `/start` or any message
3. Link your Google account when prompted

## Natural Language

Just describe what you need:

```
"Show me the listing for 123 Main Street"
"Schedule a showing for tomorrow at 2pm"
"Generate an MLS description for the new listing"
"How are my listings performing this month?"
```

## Tips
- **Be specific** — Include addresses, dates, details
- **Chain requests** — Ask for multiple things at once
- **Iterate** — Ask for adjustments if needed
