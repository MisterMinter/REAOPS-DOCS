# Using the Telegram Bot

The Telegram bot is the primary way to interact with REAOPS.

## Getting Started

1. **Find the bot** — Search for your REAOPS bot on Telegram
2. **Start a conversation** — Type `/start` or just send a message
3. **Authenticate** — Link your Google account when prompted

## Available Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and see welcome message |
| `/help` | Get help with available commands |
| `/listings` | View your active listings |
| `/calendar` | See upcoming showings and events |
| `/portfolio` | Get portfolio summary |

## Natural Language Queries

The real power of REAOPS comes from natural language. You don't need to use commands — just describe what you need:

### Property Management
```
"Show me the listing for 123 Main Street"
"Find all listings in the downtown area"
"Update the price for 456 Oak Avenue to $750,000"
```

### Scheduling
```
"Schedule a showing for tomorrow at 2pm"
"Add an open house for Saturday 10am-2pm"
"Cancel the showing with the Smiths"
```

### Marketing
```
"Generate an MLS description for the new listing"
"Write Instagram posts for this week's new listings"
"Create email copy for the Smith family follow-up"
```

### Portfolio
```
"How are my listings performing this month?"
"Which properties have been on market the longest?"
"Give me a daily brief for today"
```

## Tips

- **Be specific** — Include addresses, dates, and any relevant details
- **Chain requests** — Ask for multiple things in one message
- **Iterate** — If the first result isn't right, ask for adjustments

## Troubleshooting

**Bot not responding?**
- Make sure you're chatting with the correct bot
- Try sending `/start` again
- Check that your account is authorized

**Wrong information?**
- Double-check the listing address in your message
- The Drive folder structure affects what files are found
- Ask REAOPS to clarify what it found
