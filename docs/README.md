# REAOPS Documentation

**REAOPS** is an AI-powered real estate brokerage assistant built for brokers and agents.

## Features

- **Google Drive** — List, search, move files, create Google Docs
- **Property Listings** — Search from Drive, HubSpot, or Zillow
- **Marketing** — MLS descriptions, Instagram captions, email copy
- **Calendar** — Events, showings, open houses
- **Follow-ups** — Draft email and SMS copy
- **Portfolio Analysis** — Summarize, flag stale listings, daily briefs
- **Zillow** — Import listings from public Zillow profiles

## Quick Start

1. Get access from your REAOPS administrator
2. Sign in with Google
3. Start chatting on Telegram

## Example Commands

```
"Show me my active listings"
"Create a showing for 123 Main St tomorrow at 2pm"
"Generate marketing copy for 456 Oak Ave"
```
