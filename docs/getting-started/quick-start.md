# Quick Start Guide

Get up and running with REAOPS in minutes.

## For End Users

### 1. Get Access

Contact your REAOPS administrator to get your account set up. You'll need:

- A Google account (for Drive and Calendar access)
- Telegram (recommended) or web chat access

### 2. Connect Google Account

On first use, you'll be prompted to sign in with Google. This gives REAOPS access to:

- Google Drive (full access) — for file management
- Google Calendar — for scheduling
- Gmail (compose only) — for drafting follow-ups

### 3. Start Chatting

Once connected, simply send a message to the bot describing what you need:

**Examples:**

```
"Show me my active listings"
"Create a showing for 123 Main St tomorrow at 2pm"
"Generate marketing copy for the property on 456 Oak Ave"
"Summarize my portfolio performance this month"
"Draft a follow-up email to the Smith family"
```

## For Admins

### Initial Setup

1. **Deploy the application** — See [Deployment](./deployment) for hosting instructions
2. **Create admin user** — The first user must be added directly to the database
3. **Add users** — Go to Admin → Users to add new team members
4. **Configure integrations** — Set up Google Cloud credentials, Telegram bot token, and AI provider keys

### Adding New Users

1. Go to **Admin → Users**
2. Click **Add User**
3. Enter their Google email address
4. Assign their role (Agent, Admin)
5. They can now sign in with Google
