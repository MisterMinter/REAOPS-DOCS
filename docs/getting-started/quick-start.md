# Quick Start Guide

## For End Users

### 1. Get Access
Contact your REAOPS administrator.

### 2. Connect Google
Sign in with Google for Drive, Calendar, and Gmail access.

### 3. Start Chatting

```
"Show me my active listings"
"Create a showing for 123 Main St tomorrow at 2pm"
"Generate marketing copy for 456 Oak Ave"
"Draft a follow-up email to the Smith family"
```

## For Admins

### Initial Setup
1. Deploy the application
2. Create admin user (add directly to database)
3. Add users via Admin → Users
4. Configure integrations (Google Cloud, Telegram, AI provider)

### Adding Users
1. Go to **Admin → Users**
2. Click **Add User**
3. Enter Google email
4. Assign role (Agent or Admin)
