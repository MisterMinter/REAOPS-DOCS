# Features Overview

REAOPS provides a comprehensive set of tools for real estate professionals.

## Core Capabilities

### 📁 Google Drive

List, search, move files, and create Google Docs across a shared Drive folder tree organized by listing address.

**Example queries:**
- "Find the purchase agreement for 123 Main St"
- "Create a new folder for 456 Oak Ave listings"
- "List all files in the Smith Family folder"

### 🏠 Property Listings

Search and inspect cached listings sourced from Drive folders, HubSpot sync, or Zillow profile scraping.

**Example queries:**
- "Show me all active listings"
- "Find listings with more than 3 bedrooms"
- "What's the days-on-market for 789 Pine Rd?"

### 📸 Marketing Generation

Generate MLS descriptions, Instagram captions, email subject lines, and social card copy from listing facts and hero photos.

**Example queries:**
- "Generate an MLS description for 123 Main St"
- "Write Instagram captions for the ocean view property"
- "Create email subject lines for the new listing"

### 📅 Google Calendar

List upcoming events, create showings/open houses, and add attendees.

**Example queries:**
- "Show my calendar for this week"
- "Create a showing for 123 Main St tomorrow at 2pm"
- "Add John Smith to the open house on Saturday"

### ✉️ Follow-ups

Draft email and SMS follow-up copy for leads and contacts; create calendar reminders for outreach.

**Example queries:**
- "Draft a follow-up email to the Johnson family"
- "Create a reminder to call the Smiths next week"
- "Write an SMS reminder for the showing tomorrow"

### 📊 Portfolio Analysis

Summarize active listings by status, flag high days-on-market properties, suggest priority actions, and generate a daily brief.

**Example queries:**
- "Summarize my portfolio performance"
- "Flag properties that have been on market more than 60 days"
- "Generate a daily brief for my team"

### 🔗 Zillow

Scrape public Zillow profile pages to import listing links.

**Example queries:**
- "Import the listing from this Zillow URL"
- "Scrape my Zillow profile for new listings"

:::note
Zillow scraping is best-effort — datacenter IPs are often blocked.
:::
