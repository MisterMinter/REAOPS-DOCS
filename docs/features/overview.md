# Features Overview

## 📁 Google Drive
List, search, move files, create Google Docs across Drive folder tree.

## 🏠 Property Listings
Search and inspect cached listings from Drive, HubSpot, or Zillow.

## 📸 Marketing Generation
Generate MLS descriptions, Instagram captions, email subject lines, social copy.

## 📅 Google Calendar
List events, create showings/open houses, add attendees.

## ✉️ Follow-ups
Draft email and SMS follow-up copy, create calendar reminders.

## 📊 Portfolio Analysis
Summarize listings, flag high days-on-market, generate daily briefs.

## 🔗 Zillow
Scrape public Zillow profiles to import listing links.
